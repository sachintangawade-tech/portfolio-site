# Personal Portfolio Website (HTML & CSS)

This is a simple and clean personal portfolio website built using basic HTML and CSS.  
The website includes sections like About, Skills, and Contact.  
It is designed for beginners and freshers to showcase their profile online.

---

## Features
- Simple and clean responsive layout
- About Me section
- Skills section
- Contact information
- Easy to modify and customize

---

## Technologies Used
- HTML5
- CSS3
